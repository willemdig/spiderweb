Spiderweb-visualisering av Digdirs fellesløsninger

Nettverksvisualisering av hvordan sentrale fellesløsninger henger sammen. Figuren viser avhengigheter og samspill, ikke nødvendigvis full teknisk arkitektur.

Kategorier

Spiderweb-look
Redigerbar datamodell
GitHub Pages-klar

Korleis du redigerer figuren

Øvst i skriptet finn du NODES og EDGES. Legg til, fjern eller endre namn der for å oppdatere visualiseringa.

Kvar relasjon blir lesen som: frå node → til node, altså at den første brukar eller er avhengig av den andre.

Tips: Bruk denne som index.html i eit GitHub-repo og slå på GitHub Pages.
